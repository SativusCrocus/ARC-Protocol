# ARC Goose Logo

**File:** `logo.svg` (to be generated — this is the design brief)

## Concept

A combined mark that fuses the ARC Protocol identity with the Goose silhouette.

## Design

- **Primary shape:** A stylized goose in flight, wings spread, facing right.
- **Body:** The goose's body is formed from three interlocking hash-chain
  links, rendered as rounded rectangles in a descending diagonal — representing
  the provenance chain.
- **Eye:** A single orange Bitcoin-orange (`#F7931A`) dot. Circular. The only
  non-monochrome element.
- **Trailing wake:** Below the goose, three small dots fade out, suggesting
  records left behind.
- **Line weight:** 2px strokes, rounded caps and joins.
- **Style:** Minimal, monoline, inspired by Swiss-modern tech logos
  (Linear, Vercel). No gradients, no shadows.

## Colors

- Default foreground: `#111111` (near-black) for light backgrounds
- Inverted: `#FAFAFA` for dark backgrounds
- Accent (eye only): `#F7931A` (Bitcoin orange)

## Usage

- Square lockup: 512x512 SVG, centered with 64px padding
- Horizontal lockup: goose mark + "ARC GOOSE" wordmark in
  `Inter` or `IBM Plex Sans`, tracking +2%

## Generation

Run `distro/arc-goose/branding/generate-logo.sh` once an illustrator
or AI image pipeline is wired up. Until then this file serves as the brief.
